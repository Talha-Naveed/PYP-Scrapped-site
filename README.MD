# How to Run

Create a Virtual Environment and install the requirements.txt file

## Linux Demo

```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
$ python3 h1_finder.py
```

## Windows Demo

```bash
$ python -m venv venv
$ .\venv\Scripts\activate
$ pip install -r requirements.txt
$ python main.py
```
